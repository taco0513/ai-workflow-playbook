# 📚 Context7 활용법

## 🌟 Context7란?

**실시간 라이브러리 문서 접근 시스템**입니다. AI가 항상 최신 문서를 참조하여 정확한 코드를 생성합니다.

### 핵심 기능
- 📖 공식 문서 실시간 접근
- 🔄 항상 최신 버전 참조
- 🎯 정확한 API 사용법
- 💡 베스트 프랙티스 적용

## 🚀 Context7 작동 원리

### 자동 활성화 조건
```
1. 외부 라이브러리 import 감지
2. 프레임워크 관련 질문
3. API 사용법 문의
4. 라이브러리 특정 기능 요청
```

### 처리 과정
```
사용자 요청
    ↓
라이브러리 감지
    ↓
Context7 문서 검색
    ↓
최신 정보 반영
    ↓
정확한 코드 생성
```

```

"[    ] 정보를 한 곳에서 보고 싶다"
```

## 🚀 성공 공식

### 성공하는 프로젝트의 특징

#### 1. 🎯 명확한 목적
- 한 문장으로 설명 가능
- 구체적인 문제 해결

#### 2. 🤏 적절한 범위
- 핵심 기능 3-5개
- 17일 안에 완성 가능

#### 3. 💖 개인적 필요
- 내가 쓰고 싶은 앱
- 주변 사람들도 필요

#### 4. 📈 확장 가능성
- 기본 버전 먼저
- 나중에 기능 추가

```markdown
### 핵심 기능

3. [기능 3]

- [ ] 기본 기능 작동

### 일정
- Week 1: 기본 구조

```

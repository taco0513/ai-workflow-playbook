## 🚨 일반적인 문제 해결

### 설치가 안 될 때
```
Q: "다운로드가 안 돼요"
A: 브라우저를 Chrome으로 바꿔보세요

Q: "설치 중 에러가 나요"
A: 관리자 권한으로 실행해보세요
```

### 로그인 문제
```
Q: "구글 로그인이 안 돼요"
A: 팝업 차단을 해제하세요

Q: "로그인 후 하얀 화면이에요"
A: Cursor를 재시작하세요
```
